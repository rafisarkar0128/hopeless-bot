Hopeless Bot

General Information:

Module Aliases:

The project uses module aliases for easier imports:

  • @root: .
  • @src: src/
  • @lib: src/lib/
  • @config: src/config/
  • @structures: src/structures/
  • @handlers: src/handlers/
  • @helpers: src/helpers/
  • @schemas: src/schemas/
  • @commands: src/commands/
  • @assets: src/assets/
  • @events: src/events/

Node.js Engine Requirement:

Requires Node.js version 20 or higher.

Scripts:

  • test: Placeholder for running tests.
  • dev: Runs the bot in development mode using nodemon.
  • trace: Runs the bot with trace warnings enabled.
  • start: Starts the bot.
  • destroyGlobal: Runs a script to destroy global commands.
  • destroyGuild: Runs a script to destroy guild-specific commands.
  • lavalink: Starts Lavalink server.
  • docker-up-lavalink: Starts Lavalink server using Docker.
  • docker-kill-lavalink: Stops the Lavalink server running in Docker.
